# twitter_bot
<h2>はじめに</h2>

知り合いに頼まれて作成しました。  
が、Twitterは自動の"いいね"を禁止してるらしいです。(完成後に知りました)   
せっかく作ったのでここに供養します。

<h2>概要</h2>

このスクリプトは以下の機能を定期実行します。  

- 指定したワードを含むツイートへのいいね
- 片思いフォロワーへのフォロー
- フォローしてくれないアカウントのフォロー解除

※ 各機能のオン、オフや実行間隔、いいね数、フォロー数等は指定できます。

<h2>実行に必要な準備</h2>

- 実行環境の構築(Python)
- 設定ファイルの作成(JSON)
